# 🏡 WonderLust

**WonderLust** is a full-stack vacation rental platform inspired by Airbnb. It enables users to discover unique places to stay, create and manage property listings, and share their experiences through reviews. The application is built with a secure backend, responsive UI, and RESTful architecture.

---

## ✨ Features

* 🔐 User Authentication (Sign Up, Login & Logout)
* 🏠 Create, Edit and Delete Property Listings
* ⭐ Add Reviews and Ratings
* 📍 Browse Listings with Detailed Information
* 🛡️ Secure User Sessions and Authorization
* ✅ Server-side Form Validation using Joi
* 📱 Responsive Design with Bootstrap
* 💬 Flash Messages for Better User Experience

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Anurag2327/WonderLust.git
cd WonderLust
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Create a `.env` File

Add the following variables with **your own values**:

```env
ATLASDB_URL=your_mongodb_connection_string
SECRET=your_session_secret
```

> ⚠️ Never commit your `.env` file or any credentials to GitHub.

### 4. Run the Application

```bash
npm start
```

or

```bash
nodemon app.js
```

Visit:

```text
http://localhost:8080/listings
```
